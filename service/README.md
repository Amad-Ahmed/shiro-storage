## Installation

```bash
yarn install
```

## Environment

Add these to the .env at the root of this folder

```shell
PORT=
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
